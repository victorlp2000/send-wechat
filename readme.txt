有关发送文章的程序操作这里说明一下
  • 选取生成：设置每隔10分钟扫描一次网页，如果标题或链接地址是最近两天没有出现过的，就作为
  新文章生成图片存下来。如果网址不变但标题改了，会标示为“修改更新”。直播类的文章如果标题和
  连接不变，仅换内容，每隔3小时重新生成一次。去掉了网页内的广告和其他附加推荐评论等，只保留
  文章本身。在文章顶部附加有来源、生成时间及网址信息
  • 发送微信：正常情况下，网页新发布的文章会在15分钟内发送到群里，但因网页微信常会有不明原
  因的退出，重新启动需要用手机交互，有时会有耽搁，如文章有时间敏感，参考顶部的截屏生成时间
  • 信息源：主要是英国BBC和德国之声两个中文网，包括了出现在头条的所有文章 ，可以感受媒体的
  信息特点，也能相互印证；纽约时报则是取《简报》提供一些简单信息。在言论自由的环境下，这类
  大媒体可信度应该好一些。如能推荐其他重事实少宣扬观点的信息源，或其他改进建议更好
